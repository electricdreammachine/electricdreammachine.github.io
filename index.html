What I'm building
A React Native app to display upcoming bin collections across Somerset, and a Go API to feed it data.
Why
The council do what they can, and have recently (2020) upgraded their interface for getting up-to-date collection info on the web, but it could be a lot more convenient.
Where I'm up to with it
Got the fundamentals of the app and API down, both just need finishing up.
The API is deployed onto a Heroku free dyno, probably need to move off that because I'm going to want to do some stuff with Redis soon and I don't want to be too locked into the Heroku ecosystem by the time I'm done with that (spendy).
I've got a design I want to execute on the app, which I have basic layout in place for, just need to get some graphics together and smash it out.
